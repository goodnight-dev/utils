# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report them privately through GitHub's
[private vulnerability reporting](https://github.com/goodnight-dev/utils/security/advisories/new):

1. Go to the **Security** tab of this repository.
2. Click **Report a vulnerability**.
3. Fill out the advisory form with as much detail as you can — the affected
   package and version, reproduction steps, and the impact.

You'll get a response as soon as possible, and we'll coordinate a fix and
disclosure with you privately before anything is made public.

## Supported versions

This project is pre-1.0 and under active development. Security fixes land on the
latest released version of each package. Until the packages reach 1.0, pin to a
known-good version and watch releases.
